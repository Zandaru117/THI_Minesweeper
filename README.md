# Сапёр на C++

Полноценный сапёр, в котором соблюдены все правила игры, написанные на языке программирования C++ с использованием фреймворка Qt creator.

## 💣Режимы игры
1. **Новичок**
   ```bash
   Поле 9х9 с 10 минами.
2. **Любитель**
   ```bash
   Поле 16х16 с 40 минами на поле.
3. **Эксперт**
   ```bash
   Поле 30х16 c 99 минами, настоящее испытание для ветеранов.
4. **Особый**
   ```bash
   Настройте сами свою игру, регулируйте размеры и количество мин на поле.

Программа предназначена для операционной системы Windows.
Язык программирования c++.

## 🛠️Установка и сборка проекта

Проект разделен на два независимых репозитория для реализации архитектуры MVC:
THI_Minesweeper_dll — динамическая библиотека с бизнес-логикой игры.
THI_Minesweeper — графический интерфейс пользователя на фреймворке Qt.
Папки обоих репозиториев должны располагаться в одной общей директории для корректной работы относительных путей:

2. **Выполните сбокру проекта**

3. **Склонируйте репозиторий с графической оболочкой в соседнюю папку:**
   ```bash
   git clone https://github.com/Zandaru117/THI_Minesweeper.git
4. **Соберите проект в Qr creator**

## 🚩Особенности реализации

1. Динамическое поле: Игрок может вручную настраивать размеры игрового поля и создавать кастомные уровни сложности.
2. Умный лимит на количество бомб: Игра автоматически ограничивает максимальное число мин под размер выбранного поля. Это нужно для того, чтобы игрок физически не мог забить минами всё поле целиком и у программы всегда оставалось свободное место для генерации безопасной зоны вокруг первого клика.
3. Безопасный первый ход: Генерация мин происходит только после первого клика игрока. Это гарантирует, что первый ход никогда не приведет к подрыву, а вокруг первой открытой ячейки всегда будет свободная зона 3х3.
4. Разделение логики и интерфейса (MVC): Вся математика игры (генерация, проверки, подсчет мин) вынесена в отдельную DLL-библиотеку, а Qt-приложение отвечает только за красивое отображение и клики.

## 🎮Управление

1. **Левая и правая кнопки мыши (ЛКМ и ПКМ):**
    ```bash
    Левая кнопка мыши (ЛКМ): * Клик по закрытой ячейке — вскрывает её. Если там мина — игра окончена. Если ячейка пустая, автоматически откроются все соседние безопасные зоны.
    Правая кнопка мыши (ПКМ): * Клик по закрытой ячейке — устанавливает или снимает флаг (🚩). Используется для пометки мест, где гарантированно находится бомба. Клетки с флагами защищены от случайного нажатия ЛКМ.
   
2. **Двойной клик или клик по цифре (Аккорд):**
    ```bash  
    Если вокруг открытой цифры уже расставлено нужное количество флагов, клик по этой цифре автоматически вскроет все остальные соседние ячейки. Это значительно ускоряет очистку поля.
